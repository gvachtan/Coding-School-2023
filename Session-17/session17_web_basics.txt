’σκηση - Session-17 - Coding school



1. Ένας συνηθισμένος τρόπος για να υποβληθεί αίτημα σ' έναν εξωτερικό API χωρίς να γνωρίζουμε 
ολόκληρη τη διεύθυνση URL είναι να χρησιμοποιήσουμε ένα κλειδί API. 
Το κλειδί API μεταβιβάζεται ως παράμετρος στη διεύθυνση URL.

2. Εάν υποβληθεί αίτημα GET και ζητηθεί μια εικόνα JPEG, ο τύπος περιεχομένου 
στην απάντηση θα πρέπει να είναι "image/jpeg".

3. Για να γίνει ένα αίτημα GET και να αποθηκευτεί προσωρινά το περιεχόμενό του για 1 ημέρα, 
μπορεί να οριστεί η κεφαλίδα "Cache-Control" με τιμή "max-age=86400".

4. Ναι, γίνεται μια εφαρμογή να κάνει μόνο αιτήματα GET. Τα αιτήματα GET 
χρησιμοποιούνται για την ανάκτηση πληροφοριών από έναν διακομιστή.

5. Ναι, μια εφαρμογή μπορεί να κάνει μόνο αιτήματα POST. Τα αιτήματα POST 
χρησιμοποιούνται για την υποβολή δεδομένων σε έναν διακομιστή.

6. Εάν δεν έχει σταλθεί ένα υποχρεωτικό πεδίο σε ένα αίτημα API, 
αναμένεται ένας κωδικός κατάστασης 400 Bad Request.

7. Εάν δεν υπάρχει επικοινωνία με τη βάση δεδομένων, περιμένουμε 
έναν κωδικό κατάστασης 500 Internal Server Error.

8. Όχι, δε γίνεται να συνδεθείς με αίτημα GET. Τα αιτήματα GET χρησιμοποιούνται 
για την ανάκτηση πληροφοριών και δεν προορίζονται για την τροποποίηση δεδομένων στο διακομιστή. 
Η σύνδεση απαιτεί συνήθως την αποστολή ευαίσθητων πληροφοριών, η οποία πρέπει να γίνεται μέσω 
ενός ασφαλούς αιτήματος, όπως ένα αίτημα POST.

9.Η διαφορά μεταξύ PUT και POST είναι ότι το PUT χρησιμοποιείται για την 
ενημέρωση ενός υπάρχοντος πόρου στον διακομιστή, ενώ το POST χρησιμοποιείται 
για τη δημιουργία ενός νέου πόρου στον διακομιστή.

10. Για να ανεβεί ένα αρχείο, θα πρέπει να χρησιμοποιηθεί η μέθοδος POST με το 
χαρακτηριστικό "enctype" να έχει οριστεί σε "multipart/form-data".
